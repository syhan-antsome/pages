📋 프로젝트 개요

AKC B2C는 아마노코리아의 B2C(Business-to-Consumer) 주차장 서비스를 위한 마이크로서비스 아키텍처(MSA) 기반 플랫폼입니다. 사용자 중심의 주차장 관리, 결제, 알림 시스템을 클라우드 네이티브 환경에서 확장 가능하고 유지보수가 용이한 구조로 개발하고 있습니다.

핵심 목표

  1. 확장성: 마이크로서비스 아키텍처를 통한 서비스별 독립적 확장
  2. 가용성: 컨테이너 기반 배포로 높은 가용성 보장
  3. 사용자 경험: 직관적이고 편리한 주차장 서비스 제공
  4. 통합성: 소셜 로그인, 외부 결제 시스템 등 다양한 서비스 통합

🏗️ 시스템 아키텍처

🚀 서비스별 개발 현황

1. AuthService (인증 서비스) - 진행 중 80%

주요 기능:

  • ✅ 이메일/비밀번호 로그인 및 회원가입
  • ✅ JWT 토큰 관리 (생성, 검증, 갱신, 무효화)
  • ✅ 소셜 로그인 (Google, Kakao, Naver)
  • 🔄 비밀번호 재설정 및 회원 탈퇴
  • 🔄 결제 이력 관리

2. MemberService (회원 관리 서비스) - 진행 중 70%

주요 기능:

  • ✅ 회원 정보 관리 (CRUD)
  • ✅ 회원 목록 조회 (페이징, 검색, 정렬)
  • ✅ 회원 상태 관리 (활성, 비활성, 정지)
  • 🔄 법인웹 회원 정보 관리
  • 🔄 통계 데이터 및 관리자 대시보드

3. AdminService (관리자 서비스) - 진행 중 80%

주요 기능:

  • ✅ 관리자 인증 및 권한 관리
  • ✅ 관리자 계정 관리
  • 🔄 시스템 모니터링
  • ✅ 사용자 관리 기능

4. ParkingService (주차장, 주차권 관리 서비스) - 진행 중 50%

주요 기능:

  • ✅ 주차장 정보 관리
  • ✅ 주차 공간 상태 관리
  • ✅ 주차장 검색 및 조회
  • 🔄 주차권 구매관리 기능
  • 🔄 내부 주차권 검색 및 조회
  • ✅ 내부주차장 검색 및 조회
  • ✅ 이미지 및 메타데이터 관리
  • 🔄 kakfa 연동 테스트

5. NotificationService (알림 서비스) - 진행 중 90%

주요 기능:

  • ✅ 이메일 알림 (Gmail SMTP 연동)
  • ✅ 푸시 알림 (Firebase 연동)
  • 🔄 Kafka 이벤트 기반 알림
  • ✅ 알림 이력 관리
  • 🔄 알림 연동 테스트

6. PaymentService (결제 서비스) - 진행 중 50%

주요 기능:

  • ✅ 결제 게이트웨이 연동 (설계 완료)
  • 🔄 결제 이력 관리
  • 🔄 환불 처리 시스템
  • 🔄 구독 관리

7. CouponService (쿠폰 서비스) - 진행 중 40%

주요 기능:

  • ✅ 쿠폰 생성 및 관리 (설계 완료)
  • 🔄 할인 정책 엔진 및 스케쥴
  • 🔄 쿠폰 사용 이력

🛠️ 기술 스택

Backend

  • Framework: Spring Boot 3.2.6
  • Language: Java 17
  • Build Tool: Gradle 8.14.1
  • Database: MySQL 8.0
  • Cache: Redis 7
  • Message Queue: Apache Kafka 7.4.0
  • Security: Spring Security + JWT
  • Documentation: SpringDoc OpenAPI 3

Infrastructure

  • Containerization: Docker & Docker Compose
  • Reverse Proxy: Nginx Alpine
  • Service Discovery: DNS 기반 서비스 발견
  • Orchestration: Docker Compose (개발환경)

External Integrations

  • 소셜 로그인: Google OAuth 2.0, Kakao API, Naver API
  • 이메일: Gmail SMTP
  • 푸시 알림: Firebase Cloud Messaging (FCM)
  • 결제: 추후 PG사 연동 예정

📋 결론

AKC B2C 프로젝트는 현재 약 60% 완료 상태로, 핵심 인프라와 주요 서비스들이 안정적으로 구현되었습니다. 마이크로서비스 아키텍처의 장점을 활용하여 확장성, 유지보수성, 안정성을 모두 확보한 상태입니다.

특히 AuthService와 MemberService의 완성도가 높아 사용자 관리 기능이 완전히 작동하며, 소셜 로그인 기능까지 포함하여 현대적인 사용자 경험을 제공할 준비가 되었습니다.

향후 예약, 결제, 쿠폰 시스템을 순차적으로 완성하여 2025년 말까지 완전한 주차장 서비스 플랫폼을 구축할 예정입니다.
